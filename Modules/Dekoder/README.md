# Moduł Dekoder

## Przydatne funkcje: <br>
Instalacja modułu:
```
cd path/to/module/Dekoder
pip install .
```
Uruchomienie jako kontenera:
```
cd path/to/module/Dekoder
sudo docker build dekoder:v1.0 .
sudo docker run dekoder:v1.0 <komenda>
```
Na przykład:
```
sudo docker run dekoder:v1.0 to_bin 1234
```
